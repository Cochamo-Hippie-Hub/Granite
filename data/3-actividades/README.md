## Como contribuir

1. Crea una carpeta en la subcategoria correspondiente
2. Incluye un `README.md` con las 7 capas transversales
3. Archivos .gpx para tracks, JPEG/PNG para topos
4. Envia un Pull Request o escribe a cochamohippiehub@gmail.com

**Etica de publicacion:**
- No publicar topos de rutas que los aperturistas piden mantener sin documentar
- Indicar claramente si la informacion esta verificada o es aproximada
- Senalar peligros objetivos conocidos
