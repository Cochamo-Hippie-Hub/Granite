# Infraestructura

Documentación de la infraestructura física del área protegida: senderos,
señalética, saneamiento, conectividad, energía y edificaciones.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `senderos-caminos/` | Inventario de senderos, estado, mantención, trazados GPS, puentes, vados |
| `senaletica/` | Sistema de señalética, diseños, ubicaciones, mantención, estándares |
| `saneamiento/` | Baños secos, manejo de residuos, puntos de reciclaje, compostaje, agua potable |
| `conectividad/` | Puntos de internet (Starlink), cobertura de radio, redes LoRa, zonas sin señal |
| `energia/` | Sistemas de energía: solar, micro-hidro, baterías, puntos de carga |
| `edificaciones/` | Refugios, centros de visitantes, bodegas, casas de guardaparques |

Fotografías de estado con fecha y coordenadas son especialmente valiosas. Trazados .gpx de senderos son el formato ideal.
