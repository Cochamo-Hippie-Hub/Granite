# Visitantes

Gestión de visitantes al área protegida: educación ambiental, capacidad
de carga, sistema de reservas, interpretación del patrimonio y voluntariado.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `educacion-ambiental/` | Materiales educativos, guiones de charlas, fichas de especies |
| `capacidad-carga/` | Estudios de capacidad de carga, límites por sector, datos de afluencia |
| `reservas/` | Sistema de reservas de campamentos, documentación del proceso actual |
| `interpretacion/` | Guiones interpretativos, paneles informativos, recorridos temáticos |
| `voluntariado/` | Programa de voluntariado, perfiles de cargo, protocolos, registro |

Los materiales educativos en formatos editables (Markdown, SVG) son más útiles que PDFs cerrados. Los registros de afluencia se publican en CSV con datos agregados — no datos personales de visitantes.
