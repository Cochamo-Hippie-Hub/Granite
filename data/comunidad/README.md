# Comunidad

Información sobre las comunidades que habitan el territorio protegido y su
entorno: familias locales, arrieros, pueblos originarios, actividades
económicas y procesos de participación.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `familias-locales/` | Registro de familias del territorio, actividades productivas, necesidades |
| `arrieros/` | Red de arrieros, rutas de arrieraje, calendarios, protocolos, patrimonio ecuestre |
| `pueblos-originarios/` | Comunidades indígenas vinculadas al territorio, derechos territoriales |
| `desarrollo-economico/` | Turismo comunitario, emprendimientos, cadenas de valor |
| `participacion/` | Procesos de participación ciudadana, talleres, consultas, actas, acuerdos |

## Nota ética

Esta sección existe para preservar y hacer visible el conocimiento y la
voz de las comunidades del territorio. No es un repositorio de datos
extractivos sobre comunidades. El principio rector es: las comunidades
son autoras, no objetos de estudio.

**Consentimiento:** toda información que identifique a personas o familias
requiere su consentimiento previo antes de ser publicada.
