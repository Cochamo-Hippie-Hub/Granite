# Geomorfologia granitica del sector Anfiteatro

> **Este es un estudio ficticio creado como modelo.** Muestra como
> estructurar una contribucion cientifica con las 7 capas transversales.

- **Autores:** Dra. Elina Korhonen, Dr. Matias Fernandez, Dr. Jukka Virtanen

## Resumen

Levantamiento fotogrametrico y analisis estructural de las paredes
graniticas del sector Anfiteatro. Modelo digital de superficie de alta
resolucion (5 cm/pixel) cubriendo 2,4 km2. Se identificaron tres familias
principales de diaclasas que controlan la morfologia de las paredes.

## Contenido

- `datos/puntos-control-gnss.csv` — 5 puntos de control terrestre
- `datos/diaclasas-mediciones.csv` — 5 mediciones de orientacion
- `resultados/familias-diaclasas.csv` — 3 familias estructurales
