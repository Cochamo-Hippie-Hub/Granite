# 1. Conservacion y sistemas ecologicos

Base cientifica y ecologica del territorio.

| Subcarpeta | Contenido |
|---|---|
| `conocimiento/` | Geologia, flora, fauna, hidrologia, clima |
| `monitoreo/` | Camaras trampa, sensores, satelite, terreno |
| `gestion-ecologica/` | Planes de manejo, restauracion, especies prioritarias, invasoras |
| `riesgos/` | Incendios, eventos climaticos, remociones, amenazas |

## Como contribuir

1. Crea una carpeta en la subcategoria correspondiente
2. Incluye un `README.md` con las 7 capas transversales (ver `data/README.md`)
3. Envia un Pull Request o escribe a cochamohippiehub@gmail.com

Ver `conocimiento/ejemplo-estudio-modelo/` para un modelo completo.

**Informacion sensible:** ubicaciones exactas de especies en peligro critico
deben generalizarse (coordenadas aprox. a 1 km).
