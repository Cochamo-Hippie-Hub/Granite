# 5. Comunidad, historia y patrimonio

Dimension humana y cultural.

| Subcarpeta | Contenido |
|---|---|
| `habitantes/` | Familias, arrieros, residentes |
| `practicas/` | Uso del territorio, tradiciones productivas |
| `historia/` | Ocupacion, relatos, hitos |
| `patrimonio/` | Arqueologia, rutas historicas, cultura material |

## Como contribuir

1. Crea una carpeta en la subcategoria correspondiente
2. Incluye un `README.md` con las 7 capas transversales
3. Envia un Pull Request o escribe a cochamohippiehub@gmail.com

**Consentimiento:** informacion que identifique personas o familias
requiere su autorizacion previa.

**Voz propia:** se privilegian materiales generados por las propias
comunidades, no interpretaciones externas.

**Sitios arqueologicos:** no publicar coordenadas exactas sin consultar
al Consejo de Monumentos Nacionales.
