# Territorio

Información geoespacial y de planificación territorial: cartografía,
límites prediales, uso de suelo, cuencas hidrográficas y zonificación.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `cartografia/` | Mapas base, cartas topográficas, modelos digitales de elevación, ortomosaicos |
| `limites/` | Deslindes del área protegida, límites comunales, predios colindantes |
| `uso-suelo/` | Clasificación de uso de suelo, cobertura vegetal, áreas intervenidas |
| `cuencas/` | Delimitación de cuencas y subcuencas, red hídrica, puntos de monitoreo |
| `zonificacion/` | Zonas de preservación, uso público, uso especial, recuperación |

## Estándares cartográficos

- **Datum:** WGS84
- **Proyección para datos locales:** UTM Zona 18 Sur (EPSG:32718)
- **Formato vectorial preferido:** GeoJSON (.geojson) — GitHub los visualiza como mapas
- **Formato raster:** GeoTIFF (.tif) con metadata de proyección incluida

**Nota:** los deslindes publicados aquí son referenciales. Los límites legales vigentes son los inscritos en el Conservador de Bienes Raíces correspondiente.
