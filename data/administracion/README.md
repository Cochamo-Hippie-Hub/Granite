# Administración

Gobernanza, estructura organizacional, reglamentos y políticas operativas
del área protegida.

---

## Qué contiene esta sección

- Organigrama institucional y roles
- Reglamento interno del área protegida
- Normas de comportamiento para visitantes, operadores y personal
- Políticas operativas (temporadas, horarios, tarifas si existen)
- Actas de reuniones de directorio y comités
- Planes operativos anuales
- Convenios de colaboración interinstitucional
- Protocolos de coordinación entre actores del territorio

---

## Cómo contribuir

Si trabajas en la gestión del área protegida o en instituciones relacionadas:

1. Identifica en qué subcategoría encaja tu documento
2. Crea una carpeta con nombre descriptivo (ej: `reglamento-visitantes-2026/`)
3. Incluye un `README.md` con: título, fecha, versión, autor o institución responsable
4. Envía un Pull Request o escribe a cochamohippiehub@gmail.com

**Formatos aceptados:** Markdown (preferido), PDF para documentos oficiales

---

## Cómo acceder

Navega las carpetas directamente en GitHub. Todos los documentos son públicos.

---

## Nota sobre confidencialidad

Los documentos que contengan información personal, datos financieros sensibles
o material bajo reserva institucional **no deben publicarse aquí**. Esta sección
es para documentos de gestión pública del área protegida.
