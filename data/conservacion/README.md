# Conservación

Planes de manejo, restauración ecológica, gestión de especies prioritarias
e invasoras, y prevención y control de incendios forestales.

## Subcarpetas

| Carpeta | Contenido |
|---|---|
| `planes-manejo/` | Plan de manejo del área protegida, planes de uso público, planes de acción por sector |
| `restauracion/` | Proyectos de restauración ecológica, reforestación, recuperación de hábitat |
| `especies-prioritarias/` | Fichas de especies clave: huemul, pudú, puma, alerce, rana de Darwin, cóndor |
| `especies-invasoras/` | Registro de especies exóticas, planes de control, protocolos de erradicación |
| `incendios/` | Planes de prevención, mapas de riesgo, protocolos de respuesta, historial de eventos |

## Cómo contribuir

1. Crea una carpeta con nombre descriptivo en la subcategoría correspondiente
2. Incluye un `README.md` con descripción, autoría, fecha y licencia
3. Envía un Pull Request o escribe a cochamohippiehub@gmail.com

**Información sensible:** las ubicaciones exactas de nidos, madrigueras o poblaciones de especies en peligro crítico deben ser generalizadas (coordenadas aproximadas a 1 km) para evitar perturbación.
