# 4. Base territorial, normativa e institucional

El soporte duro del sistema.

| Subcarpeta | Contenido |
|---|---|
| `cartografia/` | Mapas, limites, cuencas, capas GIS |
| `propiedad/` | Dominio, servidumbres, DRC, superposiciones |
| `normativa/` | Permisos, regulaciones, ordenanzas, legislacion |
| `institucional/` | FCP, CPP, convenios, gobernanza formal |

## Como contribuir

1. Datos geoespaciales en GeoJSON preferido (GitHub los visualiza como mapas)
2. Datum WGS84, proyeccion UTM Zona 18 Sur (EPSG:32718)
3. Incluye `README.md` con las 7 capas transversales
4. Envia un Pull Request o escribe a cochamohippiehub@gmail.com

**Nota:** los deslindes publicados aqui son referenciales. Los limites legales
vigentes son los inscritos en el Conservador de Bienes Raices.
