## Como contribuir

1. Crea una carpeta en la subcategoria correspondiente
2. Incluye un `README.md` con las 7 capas transversales
3. Envia un Pull Request o escribe a cochamohippiehub@gmail.com

**Confidencialidad:** no publicar documentos con datos personales o
informacion financiera sensible.
